Цель проекта - отработка навыков програмирования на С.

Приложение будет консольным, т.к. работа с графикой меня не интересует.
Про присутствие бд еще надо будет подумать, но для начала хочу попрактиковаться с файлами.

Что должно быть реализовано:
Сервер:
- [ ] Создание пользователей 
- [ ] Хранение данных в файле
- [ ] Хэширование паролей
- [ ] Обмен открытым ключем шифрования
- [ ] Загрузка/отправка файлов
- [ ] Обработка команд с клиента

Клиент
- [ ] Авторизация/регистрация
- [ ] Загрузка/скачиваине файлов
- [ ] Обмен открытым ключем шифрования

Команды сервера
- [ ] Просмотр онлайн пользователей (/who__online)
- [ ] Просмотр списка доступных для скачивания файлов (/view_files)
- [ ] Скачивание/отправка файлов (/download "file name", /send "file name")


По мере написания могут появиться дополнительные пункты.
